# TheVertMenthe — 3D Virtual Art Gallery

A first-person 3D virtual art gallery built with React Three Fiber, Three.js, GSAP, and Tailwind CSS.

Inspired by [TheVertMenthe by Etienne Dault-Lafon](https://thevertmenthe.dault-lafon.fr/)

## Features

- 3D gallery environment with white walls and realistic lighting
- WASD / Arrow key movement with sprint (Shift)
- Click-to-move navigation with floor hotspots
- Artwork zoom with smooth GSAP camera transitions
- Custom cursor with mix-blend-mode
- Loading screen with progress bar
- Fully responsive

## Getting Started

```bash
npm install
npm run dev
```
